## üéØ How to Use Forms

### Step 1: Access Forms
```
1. Login to G-VET System
2. Navigate to respective module (KEW.PA / KEW.PS / KEW.AH)
3. Click "Printable Forms" button (purple button)
4. Select desired form from menu
```

### Step 2: Complete Forms
```
1. Fill in all required fields (marked with *)
2. Select appropriate options from dropdowns
3. Enter numerical values where applicable
4. Add remarks/notes in comment sections
5. Review for accuracy and completeness
```

### Step 3: Print Forms
```
1. Click "Print All" button or use Ctrl+P
2. Select printer (or Save as PDF)
3. Verify print preview looks correct
4. Adjust settings if needed (margins, orientation)
5. Print or save document
```

### Step 4: Obtain Signatures
```
1. Prepared By: Staff member who completed form
2. Verified By: Supervisor who checked accuracy
3. Approved By: Manager who authorized action
4. Date and sign in designated spaces
```

### Step 5: File Documents
```
1. Make necessary copies (usually 2-3 copies)
2. Distribute to relevant departments
3. File original with supporting documents
4. Update digital records in system
5. Follow retention schedule
```

## üîí Compliance & Security

### Government Regulations
All forms comply with:
- **Peraturan Perbendaharaan Malaysia**
- **Akta Rahsia Rasmi 1972**
- **Garis Panduan Keselamatan ICT Sektor Awam**

### Security Requirements
- **Confidentiality**: All forms contain sensitive government data
- **Access Control**: Only authorized personnel may complete forms

- **Disposal**: Follow government document disposal schedule
- **Audit Trail**: All forms subject to audit inspection

### Retention Schedule
| Document Type | Retention Period |
|---------------|------------------|
| Asset Forms (KEW.PA) | Permanent (until asset disposal) |

| Livestock Records (KEW.AH) | Lifetime of animal + 3 years |

## üìä Monthly Reporting Requirements

### Asset Management (KEW.PA)

- Reports to: Asset Management Officer
- Content:
  - New asset additions

  - Disposals

- Content:

  - Low stock items

  - Deaths
  - Health issues
